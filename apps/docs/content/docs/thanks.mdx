Clawlets builds on a long list of open source projects. These are the key pieces that
make the platform possible:

- **Clawdbot / OpenClaw** — the gateway and config schema Clawlets wraps
- **nix-clawdbot (nix-openclaw)** — Nix packaging and modules
- **NixOS / nixpkgs** — reproducible builds and system management
- **nixos-anywhere** — remote NixOS installs
- **disko** — declarative disk layouts
- **sops-nix** — encrypted secrets for NixOS
- **OpenTofu** — Hetzner provisioning
- **Convex** — backend + auth for the dashboard
- **TanStack Start** — web UI framework
- **Fumadocs** — docs site framework
- **pnpm / Vite / TypeScript** — tooling foundation

Thank you to all maintainers and contributors.
